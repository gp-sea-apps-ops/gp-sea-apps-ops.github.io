## Compiling Instructions!!

1. Go to Main Directory of repository
cd [directory]

2. Find current path
pwd

3. Export path to use mvn
export PATH=[path]/apache-maven-3.6.1/bin:$PATH

4. Run the following commands to comiple and execute!
mvn compile
mvn exec:java
